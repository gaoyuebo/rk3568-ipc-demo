# 系统架构

## 总体架构

```
┌─────────────────────────────────────────────────────────────┐
│                    RK3568 开发板 (Linux)                      │
│                                                               │
│  ┌──────────┐    ┌─────────────┐    ┌──────────────────┐   │
│  │ C270 USB │───→│ V4L2 采集    │───→│ NV12 原始帧       │   │
│  │ /dev/    │    │ mmap 4 buf  │    │ (YUYV→NV12 转换)  │   │
│  │ video0   │    └─────────────┘    └────────┬─────────┘   │
│  └──────────┘                                 │             │
│                                               ↓             │
│                              ┌────────────────────────┐   │
│                              │ H264 编码               │   │
│                              │ MPP 硬编 或 x264 软编   │   │
│                              └───────────┬────────────┘   │
│                                          │                 │
│                    ┌─────────────────────┼─────────────┐   │
│                    ↓                     ↓             ↓   │
│            ┌──────────────┐    ┌──────────────┐  ┌───────┐ │
│            │ MP4 本地录像  │    │ RTSP 推流     │  │(可选) │ │
│            │ /data/*.mp4  │    │ 554 端口      │  │AI输入 │ │
│            └──────────────┘    └──────┬───────┘  └───────┘ │
│                                         │                   │
│  ┌──────────────────────────────────────┼───────────────┐  │
│  │ 控制 & 监控（独立线程）                │               │  │
│  │  · HTTP API：开始/停止录像、改分辨率  │               │  │
│  │  · 帧率统计、断流检测、自动重连        │               │  │
│  │  · 日志写文件                         │               │  │
│  └──────────────────────────────────────┘               │  │
└─────────────────────────────────────────────────────────────┘
                    │ RTSP                    │ HTTP
                    ↓                         ↓
            ┌──────────────┐          ┌──────────────┐
            │ PC / VLC     │          │ 浏览器        │
            │ 看实时直播   │          │ 控制/看状态   │
            └──────────────┘          └──────────────┘
```

## 程序结构（C 多线程）

```
main
 ├── 线程1：video_pipeline（采集→编码→录像+推流）  核心
 ├── 线程2：monitor（帧率、断流、重连、日志）      差异化
 ├── 线程3：http_server（API 控制）               可选
 └── 主线程：信号处理、优雅退出
```

## 模块说明

| 模块 | 文件 | 说明 |
|------|------|------|
| V4L2 采集 | v4l2_capture.c | mmap 4 buffer，循环 DQBUF 取帧 |
| 监控 | monitor.c | 帧率统计、断流检测、重连、日志 |
| HTTP API | http_server.c | /status、/record/start、/record/stop |
| 主程序 | main.c | 多线程调度、信号处理 |

## HTTP API（可选）

| 接口 | 功能 |
|------|------|
| GET /status | 返回 fps、是否录像、运行时长 |
| POST /record/start | 开始录像 |
| POST /record/stop | 停止录像 |
| POST /config?width=&height= | 改分辨率 |
